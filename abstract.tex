 \begin{abstract}
Данная работа посвящена изучению некоторых одномерных спиновых цепочек с замороженным беспорядком. Магнитное поле представлено независимыми случайными величинами с известным дискретным распределением вероятности.

Построено точно решение для средней энергии основного состояния для различных дискретных распределений внешнего поля. Применяемый метод не использует предел нулевой температуры.

В ходе работы получены рекуррентные соотношения, которые можно использовать для эффективного численного расчёта энергии в цепочках конечной длины. Проведён сравнительный анализ с известными методами вычисления энергии основного состояния.

В работе также показано, что средняя энергия является кусочно-линейной функцией внешнего поля; получено выражение для корреляционного радиуса цепочки.
\end{abstract}
